1. Mengapa menggunakan CSS dan JavaScript dalam pemrograman web?

CSS digunakan untuk mengatur tampilan web agar menarik dan konsisten. JavaScript digunakan untuk membuat halaman menjadi interaktif dan dinamis seperti validasi form, animasi, atau manipulasi elemen.

2. Bagaimana CSS dan JavaScript dapat diintegrasikan dalam pemrograman web?

CSS dan JavaScript dapat diintegrasikan secara inline, internal, maupun eksternal melalui tag <style>, <script>, <link>, dan <script src>.

3. Jelaskan apa saja keuntungan menggunakan JavaScript dan CSS?

Keuntungan CSS: memisahkan tampilan dan konten, mudah dipelihara, dan mendukung responsivitas. Keuntungan JavaScript: membuat web interaktif, memproses data real-time, dan meningkatkan pengalaman pengguna.

4. Jelaskan apa saja keuntungan menggunakan Bootstrap?

Bootstrap menyediakan komponen siap pakai, desain responsif otomatis, tampilan konsisten, dan menghemat waktu pengembangan.
